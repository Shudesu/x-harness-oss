# X Harness 運用マニュアル

> Xステップ代替OSS「X Harness」を **獲得装置として運用する** ためのマーケター向けマニュアル。
> OSS本体は無料。本マニュアルの一部章は、運用ノウハウとして有料配布します。

---

## このマニュアルについて

X Harness は「verify_only」設計思想に基づく X 運用自動化 OSS です。
コードは無料で公開していますが、**「どうボタンを押すと結果が出るか」までは、設計と運用知識が必要** です。

このマニュアルは、その運用ノウハウを体系化したものです。

- **無料章**: ボタンを押すと何が起きるか（What）
- **🔒 有料章**: どう設計すれば獲得できるか（How / Why）
